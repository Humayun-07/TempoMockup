Screen 15 · Error States

All Error States

6 API error codes, each with a specific UI response. API spec §5.2.

HTTP 401 — SESSION EXPIRED
Session expiredYou've been logged out. Please sign in again to continue — your plan is saved.
HTTP 403 — CALENDAR SCOPE MISSING
Calendar access revokedTempo no longer has access to your calendar. Please reconnect to continue planning.
HTTP 409 — SLOT CONFLICT AT COMMIT
Slot no longer availableYour calendar changed since the plan was generated. Gym at Tue 7am now has a conflict. Please review and pick a new time.
HTTP 422 — AI PARSE FAILURE
Couldn't read your tasksTempo couldn't extract any tasks. Try entering one task per line, or use clearer wording.
HTTP 429 — RATE LIMITED
You've planned a lot todayYou've reached today's planning limit. Come back tomorrow to generate a new plan — your tasks are saved.
HTTP 503 — AI UNAVAILABLE
Planning is temporarily unavailableOur AI is taking a short break. This usually resolves in a few minutes — your tasks are saved and nothing was lost.
